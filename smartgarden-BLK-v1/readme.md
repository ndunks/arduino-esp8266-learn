# SMART GARDEN BLKP KLAMPOK - Banjarnegara


## PIN MAPPING - ESP8266

| WemMos/NodeMCU |  Connect to   |
|----------------|---------------|
| D7             | IC 595: DS    |
| D8             | IC 595: RCLK  |
| D5             | IC 595: SRCLK |
| D4             | IR Module: S  |
| D3             | DHT22: OUT    |
| D2/SDA         | I2C LCD: SDA  |
| D1/SCL         | I2C LCD: SCL  |
| VV/5V          | I2C LCD: VCC  |
| A0/ADC         | IC 4051: COMMON |
